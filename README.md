# 🛡️ AI-Powered Network Auditor v5.0
**Lead Auditor:** Gidy  


## 📋 Overview
An enterprise-grade vulnerability assessment tool that bridges **Nmap** scanning with **Gemini 3 Flash** AI analysis. Designed for automated security auditing and rapid report generation.

## ✨ Features
* **Active Recon:** Service and version detection.
* **AI Logic:** Expert-level vulnerability interpretation.
* **Professional Reporting:** Generates audit-ready Markdown files.

## 🛠️ Setup
1. `pip install -U google-genai python-dotenv`
2. Add your API Key to a `.env` file.
3. Run: `python3 auditor.py`
