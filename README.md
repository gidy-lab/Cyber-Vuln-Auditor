# Cybersecurity Audit Tool 🛡️

## Description
This is a Python-based automation script designed to perform basic system audits. It scans for specific files and configurations, generating a Markdown report of the findings. This project was developed as part of my Cybersecurity Level 6 certification at RVTTI.

## Features
* **Automated Scanning:** Quickly identifies system files and scripts.
* **Report Generation:** Automatically creates a timestamped audit report (e.g., `Audit_Report_2026-03-05.md`).
* **Clean Output:** Designed for easy integration into security documentation.

## Files in this Repository
* `auditor.py`: The main Python script for the audit.
* `Audit_Report_...`: A sample output generated by the tool.
* `.gitignore`: Prevents temporary or unrelated files from being uploaded.

## How to Run
1. Ensure you are using **Kali Linux**.
2. Run the script using Python:
   ```bash
   python3 auditor.py
